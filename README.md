# 統一企業參訪活動預約平台
### Uni-President Visit Activity Platform

這是一個專為企業參訪活動設計的飲品預約與管理系統，結合了直覺的網頁介面與 Google Sheets 自動化統計。

## 🌟 系統特色
- **視覺化選單**：即時從 GitHub 加載星巴克系列飲品圖片。
- **自動化後端**：報名資料直接寫入 Google Sheets。
- **雙語支持**：介面提供中英文對照，適合接待國際賓客。
