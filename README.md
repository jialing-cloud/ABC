# 統一企業參訪活動專用平台
### Uni-President Visit Activity Platform

這是一個專為企業參訪活動設計的飲品預約與管理系統，結合了直覺的網頁介面與 Google Sheets 的自動化統計功能。

## 🌟 系統特色
- **精美圖文選單**：即時載入星巴克系列飲品圖片，提升視覺體驗。
- **自動化後端**：報名資料直接寫入 Google Sheets，無需手動登記。
- **雙語支持**：全介面中英文對照，適合接待國際賓客。
- **即時統計**：後端支援 API 調用，可顯示即時報名人數。
